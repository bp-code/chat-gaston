## How to use
Download the project [or clone the repo](https://github.com/bp-code/chat-gaston.git):

```sh
git clone https://github.com/bp-code/chat-gaston.git
cd chat-gaston
```

Install it and run:

```sh
npm install
create the .env file using .env-template
npm start

```
